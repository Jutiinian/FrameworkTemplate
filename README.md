# Framework Template

This is the Roblox framework base I use for myself and the projects I work on. The main goal is to keep projects maintainable as they grow, with reusable infrastructure separated from game-specific features.

The repository is structured around a small set of shared services, common modules, startup entrypoints, and UI scaffolding that I can carry between projects without rebuilding the same foundation every time.

## What Is In Here

- Some useful Data, state, market, and settings services
- Shared classes, configs, modules, utilities, and game code
- Client, server, and UI startup entrypoints
- React-based UI helpers, hooks, components, and emitter utilities
- A single example feature showing the intended feature structure

This is mostly a personal base project that happens to be public.
Left public in case anybody visiting my profile wants to use it. Although I doubt that is happening 🤣.
